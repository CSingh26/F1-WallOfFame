# F1-WallOfFame

Production Next.js app for the F1 Heritage Explorer experience.

Start locally:

```bash
npm install
npm run dev
```

Open `/f1-heritage` for the museum timeline, placards, compare dock, search, and generated 3D heritage rooms.
